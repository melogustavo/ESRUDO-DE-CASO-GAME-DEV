StarRunner 2D — WebGL

Coloque aqui o build da Unity em UnityProject/Builds/WebGL e sirva essa pasta via HTTP.

Para servir localmente:

python tools/serve_webgl.py UnityProject/Builds/WebGL 8080

Depois acesse http://localhost:8080.

Dica: Para GitHub Pages, use o workflow incluso. Ele publica o conteúdo de UnityProject/Builds/WebGL.
